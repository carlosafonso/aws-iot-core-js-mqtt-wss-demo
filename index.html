This sample shows how to connect to the AWS IoT Core message broker using MQTT over secure WebSockets (WSS), using temporary credentials granted by Amazon Cognito Identity Pools.

Note that this sample uses unauthenticated identities, meaning that no authentication occurs. In a real production application you might want to use authenticated identites instead, optionally usingn a Cognito User Pool as the identity provider.

How to use this

  1. Open your browser's developer tools and go to the Console tab.
  2. Go to the AWS IoT Core MQTT test client, in the region of your choice.
  3. Click Publish to a topic and send an arbitrary message to the topic configured in this demo (by default, my/topic).
  4. The message should appear in the browser console.
